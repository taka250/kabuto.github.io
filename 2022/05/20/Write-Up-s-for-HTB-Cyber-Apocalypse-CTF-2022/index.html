md-to-pdf is a CLI tool for converting Markdown files to PDF.Affected versions of this package are vulnerable to Remote Code Execution (RCE) due to utilizing the library gray-matter to parse front matter content, without disabling the JS engine.

Amidst Us

很酷的前端

页面有隐藏上传文件的功能。此题也给了源码，分析下重要代码。

routes.py

此题有用到flask框架中的蓝图。蓝图技术，可以帮助实现flask应用的模块划分，在许多大项目中会用到。参考flask文档中所写

蓝图的基本理念是它记录了注册到应用之后将要执行的操作。在分配请求及生成两个端点间的URL时，Flask 会把蓝图和视图函数关联起来。

routes.py创建了两个简单的蓝图并绑定了各自的函数并在main.py中注册。

回归题目，后端对上传的图片进行了透明度的变化

注意到，ImageMath.eval中参数我们是可控的。估计又是cve？

很好，直接开始抓包。

body中的background参数是可控的，那么我们直接注入。
